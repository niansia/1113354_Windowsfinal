放這裡的圖片會自動顯示在地球上對應的位置。
Drop images here and they appear on the globe at the matching location.

檔名規則 / Filename convention:
  <國家代碼>.jpg          → 該國的主圖標 (hero marker)，例如  jp.jpg, tw.jpg, fr.jpg
  <國家代碼>-<編號>.jpg   → 該國的某個文化條目，例如  jp-1.jpg, jp-2.jpg

支援格式 / Formats: .jpg .jpeg .png .webp
完整清單（每個檔名 + 建議搜尋關鍵字）請見上一層的 IMAGES.md。
See ../IMAGES.md for the full list of filenames + suggested search terms.

放好圖片後重新開啟此應用即可看到（伺服器啟動時會掃描本資料夾）。
Restart the app after adding images (the server scans this folder on launch).
